# Aha.nvim

Heavily inspired (read, code copied from), octo.nvim

## Install

Using packer:

```lua
use { 'jemmyw/aha.nvim', rocks = { 'lunajson' } }
```

# Auth

You need to have downloaded the aha client from https://github.com/aha-app/aha-cli and authed:

```sh
$ npm install -g aha-cli
$ aha auth:login
```
